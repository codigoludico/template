Ejecutar tests

Para ejecutar los tests de forma continua, muy útil si vas a trabajar en ellos, sólo necesitas ejecutar:

yarn run test:watch

En caso de que querer ejecutar los tests como parte de un entorno de integración continua o bien si sólo quieres ejecutar los tests una vez:

yarn run test
