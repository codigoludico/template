Module: engine/input/keyboard

Este módulo contiene los métodos necesarios para trabajar con el teclado exponiendo una serie de funciones útiles para comenzar la escucha de eventos, detenerla y obtener el estado actual de las teclas.

Members

<inner, constant> keys

Contiene las teclas que están presionadas como valores decimales (float 32)

<inner> target

Contiene el elemento del DOM encargado de escuchar los eventos del teclado.
